# Дизайн: Strategy Lab — 24/7 ферма стратегий на песочнице (VPS)

Дата: 2026-06-12. Одобрено в диалоге («делай все что придумал»).

## Цель
Форвард-тест торговых стратегий на sandbox T-Invest на реальных котировках MOEX,
24/7 на VPS (МСК). Продукт — **журнал и статистика**, не доходность.
Контрольные стратегии (buy&hold, random) обязательны: без них не отличить скилл от удачи.

## Безопасность
Только sandbox-домен и `TINVEST_SANDBOX_KEY` (как в sandbox_grid.py). Боевого токена
на сервере нет вообще.

## Архитектура (`lab/`, stdlib-only, Python ≥3.10)

| Модуль | Ответственность |
|---|---|
| `api.py` | call() с ретраями, to_f/quot, токен из .env |
| `journal.py` | SQLite `lab.db`: trades, equity, events |
| `strategy.py` | базовый класс `Strategy` (on_start/on_tick) + контекст с buy/sell/cancel |
| `strategies.py` | GridStrategy, BuyHoldStrategy, RandomStrategy |
| `runner.py` | главный цикл: счёт на стратегию, общий тик, изоляция ошибок |
| `report.py` | сводка из журнала: equity, P&L, max drawdown, сделки, сравнение с контролем |

## Runner
- Конфиг `lab/config.json`: стратегии+параметры, инструмент (SBER), тик 60с
  (в нерабочие часы MOEX — 300с). Лимиты: ~4 вызова × 3 стратегии в минуту ≪ 200/мин.
- Каждой стратегии — свой sandbox-счёт (создаётся при первом старте, 100к виртуальных),
  маппинг в `lab_state.json`.
- Ошибка в стратегии — событие в журнал, остальные живут. SIGTERM — graceful (снять
  заявки умеет стратегия сама при рестарте — заявки переживают рестарт через state).
- Снимок equity (totalAmountPortfolio) каждой стратегии — раз в тик.

## Уроки из sandbox_grid (учтены)
- «пропала из активных» ≠ исполнена → подтверждать `GetSandboxOrderState.lotsExecuted`;
- один экземпляр — PID-лок (на Linux os.kill(pid,0) работает);
- атомарная запись state.

## Деплой
`~/tinvest-lab/` на VPS, systemd-юнит (Restart=always), логи в journald.
Отчёт: `python3 -m lab.report` руками или по cron в файл.

## Вне скоупа v1
Веб-морда, телеграм-уведомления, несколько инструментов на стратегию, оптимизация
параметров, боевые деньги (никогда).
