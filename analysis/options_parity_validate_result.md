# Безарбитражная проверка теор-цен опционов FORTS + вскрытие конвенции

**Дата:** 2026-06-19 · `analysis/options_parity_validate.py` · READ-ONLY · опционы Si-9.26 (спот USD/RUB), экспирация 2026-09-18

## Задача
Проверить теоретические цены опционов биржи (`INSTRUMENT_VALUE_THEOR_PRICE`) **модельно-независимыми**
безарбитражными тождествами и заодно вскрыть конвенцию расчёта (как считается паритет на FORTS).
THEOR_PRICE выбран намеренно — биржа публикует её даже без сделок, что снимает проблему
ликвидности дальних страйков.

## Главная находка: конвенция вскрыта из данных
Изначальная гипотеза — опционы FORTS маржируемые (futures-style), паритет без дисконта
`C−P = F−K`, тогда `F_impl=K+C−P` обязан быть **константой** по страйкам. **Оракул её отверг:**
F_impl не константа, а **линейно растёт по K** (74.6 → 75.31 на страйках 62→83).

Линейный рост — подпись **premium-style опциона с дисконтом**: премия платится вперёд, поэтому
`C−P = D·(F−K)`, откуда `F_impl = K+C−P = D·F + (1−D)·K` — прямая по K с наклоном `(1−D)`.
Метаданные подтвердили постфактум: `paymentType=PREMIUM`, `style=EUROPEAN`, `settlement=CASH`.
То есть данные сами показали верную модель раньше, чем я свериля с флагами.

## Результаты (всё держится)
**1. Пут-колл паритет (линейная форма).** Фит `F_impl = 72.5406 + 0.03318·K`,
**R²=0.999996, резидуал σ=0.039 копейки** на 43 страйках. Биржевая поверхность удовлетворяет
дисконтированному паритету практически до округления. Один выброс (K=62.5, стухшая теор-цена)
автоотсеян по правилу |резидуал|>5σ.

**2. Извлечённые из наклона величины (вторичные оракулы):**
- дисконт-фактор `D = 1 − наклон = 0.96682`;
- подразумеваемая рублёвая ставка `r = −ln(D)/T = 13.56%` — в коридоре денежного рынка
  (ключевая ЦБ 14.5%, чуть ниже, что нормально для implied-форварда);
- форвард из опционов `F = a/D = 75.030 ₽/$` против фьючерса `75.292` (Δ=26 коп.) — расхождение
  объяснимо: теор-цены посчитаны в премаркет (~03:50), фьючерс last — с вечера пред. дня (20:49),
  плюс базис спот–фьючерс.

**3. Монотонность:** коллы убывают по K, путы растут — 0 нарушений.

**4. Выпуклость (бабочка ≥ 0):** 0 нарушений на 40 тройках для коллов и для путов.

**5. Улыбка волатильности (дисконтированный Black-76, форвард 75.03, D 0.9668):**
IV из колла и IV из пута на одном страйке совпадают с **maxΔ = 0.009 п.п.** по всем страйкам
(следствие паритета — но это и независимая про��ерка корректной инверсии). Форма — скос:
26.85% на K=73 → 25.5% на K=76.5 (типичный валютный смайл).

## Вывод
Теоретическая поверхность опционов T-Invest/MOEX внутренне согласована и безарбитражна.
Конвенция: **европейские опционы с уплатой премии вперёд → дисконтированный паритет**; наклон
F_impl по страйку = дисконт-фактор, из которого извлекаются и рублёвая ставка, и форвард,
сходящийся с фьючерсом. Практический смысл: по цепочке опционов можно локально снимать
implied-ставку и форвард, строить смайл и проверять любые опционные котировки на арбитраж,
не имея отдельного источника ставки/форварда.
