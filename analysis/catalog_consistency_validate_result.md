# Согласованность каталога и карточки инструмента

**Дата:** 2026-06-19 · `analysis/catalog_consistency_validate.py` · READ-ONLY · акции/облигации/фьючерсы

## Оракул и результат
Поля инструмента из списка (`Shares`/`Bonds`/`Futures`) должны совпадать с детальной карточкой
(`ShareBy`/`BondBy`/`FutureBy`) по тому же uid.

**136/136 полей бит-в-бит** на выборке 24 инструмента (по 8 каждого класса):
- акции: ticker, classCode, isin, lot, currency, nominal — 48/48;
- облигации: + maturityDate — 48/48;
- фьючерсы: ticker, classCode, lot, expirationDate, basicAsset — 40/40.

## Вывод
Каталог (массовый List*) и детальная карточка (*By) — один источник истины: метаданные не
расходятся между эндпоинтами. Можно безопасно брать справочные поля из списка (быстрее, один
вызов на тысячи инструментов) без перепроверки карточкой. Лимит-эффективно для скринеров
(`scripts/stock_screen.py`, `ofz_screen.py`).
