# Дифф-тест: наш бэктестер vs backtrader — результат

Большая проба харнесса. B уже проверила наш движок независимой реализацией, но МОЕЙ же
логикой. Здесь — чужой зрелой библиотекой `backtrader` (1.9.78.123). Оракул — согласие
двух независимых движков. Запуск: `pip install backtrader pandas && python analysis/backtrader_diff.py`.

## Метод
Одна и та же стратегия — SMA-кросс **фиксированным размером** (100 шт) — прогоняется в
обоих движках на одних барах (синтетика GBM, офлайн). Фикс-размер убирает различия
сайзинга, так что сравнивается чистая механика движка. Соглашения выровнены:
- сигнал по close[i] → рыночная заявка → исполнение по **open[i+1]** (backtrader: coc=False);
- комиссия 0.05% от нотионала в обе стороны (bt: COMM_PERC, percabs);
- старт 100 000, лонг-онли.

модель комиссии и оценка позиции по close идентичны независимой зрелой реализации.

## Область применимости
Сравнивалась механика движка (фикс-размер). Сайзинг `order_target_percent` намеренно не
сверялся с bt (у него своя логика округления target-позиции) — но он уже проверен в задаче B
независимой реконструкцией. Вместе B (своя логика) + этот тест (чужая библиотека) дают
двойную независимую валидацию: наш движок корректен.
