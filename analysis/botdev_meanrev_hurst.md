# MEANREV_HURST — Боллинджер-реверсия с Hurst-гейтом (кандидат, 2026-07-02)

## Гипотеза

`deep/market_regime_moex.md`: возврат к среднему работает в антиперсистентном режиме
(Hurst < 0.45). Старый `meanrev` (Boll 20/2) архивирован в ферме после Deflated
Sharpe = 0% БЕЗ режимного фильтра. Проверяем: оживает ли эдж, если разрешать вход
только при R/S-Hurst(окно 100 дневных баров) ниже порога.

## Реализация

`backtest/strategies.py` → класс `MeanRevHurst` (`name="meanrev_hurst"`, в REGISTRY).
Вход: close < нижней полосы Боллинджера(bb_n, bb_k) И Hurst(hurst_n=100) ≤ hurst_max.
Выход: close > средней полосы (без гейта — фильтруем только входы). Корзина: равный
вес frac/N тикеров, лонг-онли. `hurst_max=1.5` = гейт выключен (контроль, старый meanrev).
`hurst_n=100` ФИКСИРОВАН (порог режима не оптимизируем — `deep/walk_forward_moex.md` §5).

## Данные

7 акций lab/instruments (SBER, GAZP, LKOH, ROSN, GMKN, PLZL, CHMF), дневные свечи
через `backtest/candles.py` (sandbox, кэш). **1139 общих баров, 2022-05-24 … 2026-07-01**
— февраль-2022 не пересекаем. Комиссия 5 б.п. + слиппедж 5 б.п.

## Диагноз: почему гейт не работает

Доля баров с открытым гейтом по корзине (R/S-Hurst(100), реальный прогон):

```
тикер   баров   H<0.42   H<0.45   H<0.50  median H
SBER     1051     5.4%    10.9%    26.0%     0.562
GAZP     1051     2.6%     5.5%    16.0%     0.598
LKOH     1051     4.4%     6.7%    12.1%     0.603
ROSN     1051     2.8%     7.0%    15.9%     0.636
GMKN     1047     6.4%    10.6%    18.5%     0.584
PLZL     1044     4.2%     9.3%    20.8%     0.599
CHMF     1051     1.6%     6.0%    16.7%     0.582
```

Медианный Hurst акций MOEX пост-2022 — **0.56–0.64: рынок персистентен**, антиперсистентный
режим (H<0.45) — лишь 5–11% времени. Гейт из литературы почти всегда закрыт →
1–3 сделки на OOS-окно, оценка статистически пуста, а редкие открытия не совпадают
с прибыльными реверсиями (окно 2: −6.13% при 17 сделках). Это согласуется с
`deep/walk_forward_moex.md` §6: mean-reversion с редким фильтром требует >2 лет OOS
на 100 сделок — post-2022 истории не хватает по построению.

## Вердикт: **KILL**

| критерий | порог | факт | |
|---|---|---|---|
| OOS сквозной | > 0 | **−0.56%** | ✗ |
| DSR | > 0.5 | 0.707 | ✓ |
| бьёт buyhold (OOS) | > +7.15% | −0.56% | ✗ |
| бьёт random (OOS) | > +23.11% | −0.56% | ✗ |
| OOS ≥30 сделок/шаг | 30 | 12/17/1/3 | ✗ |

Эдж не доказан: гейт улучшает in-sample качество входов, но душит частоту сделок до
статистического нуля, а сквозной OOS отрицателен и проигрывает обоим бенчмаркам.
Гипотеза «meanrev оживает при Hurst<0.45» на дневных акциях MOEX пост-2022 не
подтверждается — рынок в этом периоде персистентен, антиперсистентных окон слишком мало.

## Воспроизведение

```bash
python analysis/botdev_meanrev_hurst.py          # весь конвейер (кэш свечей переиспользуется)
# smoke на синтетике:
python -m backtest run --strategy meanrev_hurst --params "hurst_max=1.5" --synthetic mean_revert:800:3 --slippage 0.0005
```

Верификация 2026-07-02: повторный прогон (`botdev_meanrev_hurst_run2.txt`) бит-в-бит
совпал с run1 (`diff` пуст); диагностическая таблица Hurst пересчитана независимым
скриптом — числа совпали. Smoke на синтетике `mean_revert:800:3` (гейт выключен):
11 сделок, +39.87%, Sharpe 0.91 — стратегия торгует, реверсию на подходящем ряде ловит.
