# Реконструкция индекса IMOEX из состава

**Дата:** 2026-06-19 · `analysis/index_reconstruct_validate.py` · READ-ONLY · IMOEX, 46 бумаг

## Оракулы
1. **Структурный:** сумма весов состава = 100%.
2. **Капвзвешенное тождество:** дневная доходность индекса = Σ_i (w_i · r_i), где r_i — дневная
   доходность i-й бумаги (Ласпейрес). Состав+веса из `ListIndicatives(INDEX_COMPOSITION)`,
   доходности из дневных свечей.

## Результат
- **Σ весов = 100.0000%** — точно. 46 бумаг.
- Дневная доходность IMOEX за 2026-06-18: **−1.7629%**.
- Реконструкция Σ(w·r): **−1.6958%**, покрытие весов **100.00%** (0 бумаг без данных).
- **Расхождение = 0.067 п.п.** при движении индекса −1.76% — совпадение в пределах дрейфа весов.

Крупнейшие вклады (вес × доходность) — ровно те бумаги, что тянули индекс вниз 18 июня
(тяжёлые имена с −2.7…−2.8% дали по −0.27…−0.40 п.п. вклада).

## Почему остаток 0.067 п.п. (а не ноль)
Доступны ТЕКУЩИЕ веса (на момент запроса), а корректная дневная доходность считается по весам
на НАЧАЛО дня. За день веса слегка дрейфуют (бумаги с большей доходностью набирают вес), отсюда
неустранимый из этих данных остаток. Плюс IMOEX пересчитывает free-float-капитализацию
внутридневно. 0.067 п.п. на ходе −1.76% — это ~4% относительной ошибки реконструкции, типично
для Ласпейреса на текущих весах.

## Практический вывод
Индекс полностью прозрачен: его дневное движение объясняется составом и весами почти точно.
Это даёт: атрибуцию движения индекса по бумагам (кто сколько внёс), построение синтетического
портфеля-трекера IMOEX из 46 бумаг с известными весами, оценку трекинг-ошибки. Для точной
реконструкции нужны веса на начало периода (текущие дают остаток ~дрейф весов).
