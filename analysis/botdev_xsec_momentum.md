# XSEC_MOMENTUM — кросс-секционный 12-1 моментум на акциях MOEX

**Дата:** 2026-07-02 · **Вердикт: KILL** · Воспроизведение: `python analysis/botdev_xsec_momentum.py`

## Гипотеза
Классический 12-1 моментум (ранжирование корзины по доходности за lookback со скипом
последнего месяца, топ-K равными весами, ребаланс раз в M дней) даёт эдж на ликвидных
акциях MOEX.

## Предыстория (analysis/crosssec_momentum.py, 2026-06-19)
Близкая гипотеза (dualmom = относительный + абсолютный фильтр) уже проверялась на
6 бумагах / 823 барах: **эдж OOS в 1/9 конфигураций (11%)**, вывод «эджа НЕТ». Цитата:
> «→ эджа НЕТ (положителен OOS меньше чем в 70% конфигов). dualmom лишь защищает от просадки кэшем на медвежьем рынке, но НЕ зарабатывает.»

Текущий кандидат отличается (чистый кросс-секционный без абсолютного фильтра, skip-месяц,
корзина 11 бумаг вместо 6, конвейер grid→WFA→DSR) — поэтому прогнан полный бэктест,
а не автоматический kill.

## Реализация
`backtest/strategies.py` → класс `XSecMomentum` (`name="xsec_momentum"`, в REGISTRY).
Параметры: `lookback` (окно), `skip` (пропуск последних баров), `rebalance`, `top`, `frac`.
Всегда в рынке (без кэш-фильтра): проверяем именно ранжирование.

Smoke на синтетике (`python -m backtest run --strategy xsec_momentum --params
"lookback=120,skip=10,rebalance=21,top=3" --synthetic basket:8:900:7`): 65 сделок,
ret +74.78%, Sharpe 1.23 — торгует, метрики считаются.

## Данные
11 бумаг (SBER GAZP LKOH ROSN GMKN PLZL CHMF NVTK MGNT TATN VTBR), дневные свечи через
`backtest/candles.from_tinvest` (sandbox, кэш `backtest/.cache`), days=1000 → API отдал
1129 общих баров **2022-05-24 … 2026-07-01** (после break февраля-2022 — граница не
пересечена). Комиссия 0.05% + слиппедж 0.05%, кэш 100 000 ₽.

истории только вредит.

## 2. Anchored walk-forward, 4 окна
Прогнано ДВАЖДЫ: штатный `optimize.walk_forward` и вариант с прогревом истории
(warm-up: OOS-окно получает lookback предыстории, метрики считаются только по OOS —
иначе при lookback≈189 стратегия молчит полокна, 0–6 сделок).

Штатный (run1):
```
  окно 1: OOS ret   +1.21%   окно 2: +11.02%   окно 3: -1.88%   окно 4: -15.75%
  сквозной OOS: -7.12%
```
С прогревом (run2, честнее по числу сделок):
```
  окно 1: OOS ret  +22.64%  Sharpe  +1.46  сделок  14  IS Sharpe +2.68
  окно 2: OOS ret  -15.75%  Sharpe  -0.68  сделок   9  IS Sharpe +2.03
  окно 3: OOS ret   -5.54%  Sharpe  -0.10  сделок  20  IS Sharpe +0.96
  окно 4: OOS ret  -17.88%  Sharpe  -1.15  сделок  29  IS Sharpe +0.57
  сквозной OOS: -19.85%
```
3 из 4 окон в минусе, весь плюс — в одном первом окне (концентрация >100% профита
в 1 шаге — красный флаг по deep/walk_forward_moex.md). Сделок на шаг 9–29 — ниже
порога 30, т.е. статистика слабая, но знак устойчиво отрицательный в обоих вариантах.
Деградация IS→OOS тотальная: IS Sharpe +2.68…+0.57 → OOS −0.68…−1.15.

## 3. Deflated Sharpe (32 испытания)
```
  PSR (SR>0)             = 75.0%
  Deflated SR (×32 исп.) = 50.2%
  чувствительность: mean=0.05 std=0.15 (>0 у 59%), robustness=0.31
```
DSR=0.502 — ровно на границе: лучший IS-Sharpe неотличим от артефакта перебора 32 комбо.

## 4. Бенчмарки (та же лента, те же издержки)
```
buyhold                  ret    +3.73%  Sharpe  +0.14  dd  -35.64%
random(seed=1)           ret    +9.82%  Sharpe  +0.25  dd  -18.69%  сделок 197
best IS                  ret   +21.05%  Sharpe  +0.32  dd  -30.41%  сделок 107
```
In-sample лучший конфиг бьёт оба бенчмарка — но это ПОСЛЕ выбора лучшего из 32; OOS
проверку не переживает.

## Вердикт: KILL
| Критерий | Порог | Факт | |
|---|---|---|---|
| OOS-результат | > 0 | **−19.85%** (warm-up) / −7.12% (штатный) | ✗ |
| DSR | > 0.5 | 0.502 (на грани) | ~ |
| Бьёт buyhold и random | оба | только in-sample | ✗ |

Кросс-секционный моментум на посткризисной истории MOEX (2022-05…2026-07) эджа не даёт:
IS-результат — артефакт подбора, OOS стабильно отрицательный, профит сконцентрирован
в одном окне 2023 года (общий бычий тренд). Согласуется с выводом исследования
2026-06-19 по dualmom (эдж в 1/9 конфигов). В ферму не берём.

Возможная причина: корзина из 11 сильно скоррелированных сырьевых/банковских фишек —
слишком узкий кросс-секшн для моментум-премии (в литературе она живёт на сотнях бумаг),
а посанкционный рынок MOEX ротируется быстрее, чем 3-9-месячные окна успевают поймать.
