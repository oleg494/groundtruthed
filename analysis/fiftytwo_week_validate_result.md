# 52-недельные max/min: фундаментал против свечей

**Дата:** 2026-06-19 · `analysis/fiftytwo_week_validate.py` · READ-ONLY · 7 голубых фишек

## Оракул
Поля `highPriceLast52Weeks` / `lowPriceLast52Weeks` из `GetAssetFundamentals`. Реконструкция —
экстремумы дневных свечей за последние 52 недели. Заодно вскрываем: считается по интрадей
High/Low или по ценам закрытия.

**Конвенция вскрыта: интрадей High/Low** (7/7 совпало с экстремумами High/Low свечей; 0/7 с
ценами закрытия). Все 14 экстремумов (7 high + 7 low) совпали бит-в-бит.

## Вывод
52-недельный диапазон в фундаментале = max(daily High) / min(daily Low) за год — полностью
воспроизводится из свечей. Полезно: можно строить «дистанцию от 52w-max/min» для любого окна
и инструмента локально, не дёргая фундаментал, с гарантией той же методики (интрадей экстремумы).
